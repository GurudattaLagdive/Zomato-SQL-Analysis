# Zomato Data Analysis using SQL

## Project Overview

This project analyzes a Zomato food delivery database using SQL to solve real-world business problems. The analysis covers customer behavior, restaurant performance, delivery efficiency, rider analytics, revenue trends, and customer segmentation.

## Dataset Tables

* Customers
* Orders
* Restaurants
* Deliveries
* Riders

## SQL Concepts Used

* Common Table Expressions (CTEs)
* Window Functions (RANK, DENSE_RANK, LAG)
* Aggregate Functions
* Date and Time Functions
* CASE Statements
* Subqueries
* Joins (INNER, LEFT)
* Customer Segmentation Techniques

## Business Problems Solved

1. Top 5 most ordered dishes by a customer.
2. Peak ordering time slots using 2-hour intervals.
3. Average order value for high-frequency customers.
4. Customers spending more than ₹100K.
5. Orders placed but not delivered.
6. Restaurant revenue ranking by city.
7. Most popular dish in each city.
8. Customers inactive in 2024 but active in 2023.
9. Restaurant cancellation rate comparison.
10. Average rider delivery time.
11. Restaurant month-over-month growth analysis.
12. Gold vs Silver customer segmentation.
13. Rider monthly earnings calculation.
14. Rider rating analysis.
15. Peak order day for each restaurant.
16. Customer lifetime revenue analysis.
17. Monthly sales trend comparison.
18. Rider efficiency ranking.
19. Seasonal demand analysis for dishes.
20. City revenue ranking.
